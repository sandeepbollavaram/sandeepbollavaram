## 🧠 About

I'm **Sandeep Kumar Bollavaram** — an AI Systems, Platform, and Full Stack Engineer building production software that ships.

My work spans the full depth of a system: from **custom CRDs and admission webhooks** in Go, to **MCP-based agent memory layers**, to **native Windows AI assistants** with real installer pipelines. I don't build demos — I build software designed to run in production.

Currently engineering at **[Flexdee Technologies](https://flexdee.in)** — AI platform features, backend systems, and CI/CD pipelines end to end.
